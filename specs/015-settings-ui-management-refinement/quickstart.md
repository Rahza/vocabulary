# Quickstart: UI & Settings Verification

## Verification Steps

### 1. Theme Toggling

1. Navigate to **Settings** (`/settings`).
2. Find the new **"Thema"** (Theme) selection.
3. Switch between **Light**, **Dark**, and **System**.
4. Verify the UI colors update instantly and persist on reload.

### 2. Autocomplete Check

1. Go to the **Generator** page.
2. Click into the **"Thema / Kategorie"** input.
3. Verify that the browser does not show a dropdown of previous entries.

### 3. Management UI

1. Navigate to the **Collection** (`/vocabulary`).
2. Observe the new compact header layout.
3. Verify that all actions are still accessible.

### 4. Zero Count States

1. Navigate to a category with 0 words (or clear all data in settings).
2. Verify that a playful "Empty State" component is shown with an action button.

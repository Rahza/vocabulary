# Quickstart: Generator Verification

## Verification Steps

### 1. Save Operation
1. Go to the **Generator** page.
2. Enter a topic and click "Vokabeln generieren".
3. Once the results appear, click **"Behalten!"** (Save).
4. Verify that the list smoothly fades out and the **Generator Form** reappears immediately.
5. Verify there is no "empty page" flicker.

### 2. Discard Operation
1. Generate vocabulary again.
2. Click **"Verwerfen"** (Discard).
3. Confirm the action in the dialog.
4. Verify that the UI returns to the **Generator Form** smoothly.

### 3. Individual Deletion
1. Generate vocabulary.
2. Delete items one by one using the trash icon until the list is empty.
3. Verify that once the last item is deleted, the UI automatically returns to the **Generator Form**.

# Data Model: AI Vocabulary Trainer

## Entities

### VocabularyPair

Represents a single word pair and its associated metadata.

```typescript
type UUID = string;

interface VocabularyPair {
  id: UUID;
  german: string;
  czech: string;
  mnemonic: string; // generated by AI
  tags: string[]; // e.g., ["Travel", "Food"]
  difficulty: 'Beginner' | 'Intermediate' | 'Advanced';
  createdAt: string; // ISO Date
}
```

### LeitnerState

Tracks the learning progress for a specific vocabulary pair in a specific direction.

```typescript
interface LeitnerState {
  vocabId: UUID;
  direction: 'DE_TO_CZ' | 'CZ_TO_DE';
  box: 1 | 2 | 3 | 4 | 5;
  lastReviewed: string; // ISO Date
  nextReview: string; // ISO Date
  history: ReviewResult[];
}

interface ReviewResult {
  date: string; // ISO Date
  success: boolean;
}
```

### UserSettings

Stores app preferences.

```typescript
interface UserSettings {
  openaiApiKey?: string;
  theme: 'light' | 'dark' | 'system';
  dailyGoal: number; // number of cards
}
```

### TagStats

Aggregated statistics for a tag.

```typescript
interface TagStats {
  tagName: string;
  totalWords: number;
  masteredWords: number; // Box 5
  learningWords: number; // Box 1-4
}
```

## Storage Schema (LocalStorage)

The data will be stored in LocalStorage under the following keys:

- `vocab_data`: `VocabularyPair[]`
- `leitner_state`: `LeitnerState[]`
- `user_settings`: `UserSettings`

# Quickstart: Playful Design Revision

## New Dependencies

Ensure `framer-motion` and `lucide-react` are installed (installed in Feature 001).

## Font Setup

To use the playful typography:

1. Import `Quicksand` in `src/app/layout.tsx` via `next/font/google`.
2. Update the `tailwind.config.ts` font-family section.

## Theme Application

Primary classes to use for the new look:

- `bg-playful-indigo`
- `rounded-playful`
- `shadow-playful-button` (custom utility)

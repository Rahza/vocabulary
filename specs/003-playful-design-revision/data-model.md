# Data Model: Playful Design Revision

## Entities

### ThemeTokens (Runtime only)

Used to define the playful aesthetic tokens within Tailwind and component props.

| Token           | Type   | Value       |
| --------------- | ------ | ----------- |
| `playfulIndigo` | Color  | `#6366f1`   |
| `playfulYellow` | Color  | `#fbbf24`   |
| `playfulGreen`  | Color  | `#34d399`   |
| `playfulRed`    | Color  | `#f87171`   |
| `baseRadius`    | string | `20px`      |
| `shadowDepth`   | string | `0 4px 0 0` |

## Persistence

No changes to `VocabularyPair` or `LeitnerState` persistence.
